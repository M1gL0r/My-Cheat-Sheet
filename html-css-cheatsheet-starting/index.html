CSS Selectors and Specificity Cheat Sheet

A selector is used to target the specific HTML element(s) to be styled by the declaration.

SELECTORS MEANING EXAMPLE
Universal Selector The universal selector selects all elements of any type. * {}
Targets all elements on the page.
Type Selector The type selector matches the type of the element in the HTML document. p, h1, h2 {}
Targets the <p>, <h1>, <h2> elements.
Class Selector Matches an element whose class attribute has a value that matches the one specified after the period (full stop) symbol. .row {}
Targets any element whose class attribute has a value of row.
p.note {} Targets only <p> elements whose class attribute has a value of note.
ID Selector If an HTML element needs to be styled uniquely, we can give it an ID using the id attribute. # dogs{}
Targets the element whose id attribute has a value of dog.

You can add ! important after any property value to indicate that it should be considered more important than other rules that apply to the same element.

Specificity is the order by which the browser decides which CSS styles will be displayed. A best practice in CSS is to style elements while using the lowest degree of specificity so that if an element needs a new style, it is easy to override. IDs are the most specific selector in CSS, followed by classes, and finally, type.
